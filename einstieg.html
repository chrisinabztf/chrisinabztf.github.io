Aufbau einer Webseite

  1. Der Aufbau einer Webseite besteht aus zwei wichtigen Bereichen:

    1. Head: Im "Kopf" werden grundlegende Dinge angegeben. So bestimmt man hier zum Bespiel den Titel oder den Autor einer Webseite.
    2. Body: Im "Körper" der Webseite wird der Inhalt einer Webseite abgefüllt. Hier kommen also die Text und die Bilder rein.

  2. Der Aufbau sieht also graphisch so aus:

    <html>
    <head>
    <title>Page title</title>
    </head>
    <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
    <p>This is another paragraph.</p>
    </body>
    </html>
